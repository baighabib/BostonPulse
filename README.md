# BostonPulse
Predictive Analytics for Housing Markets

BostonPulse is a data analytics and machine learning project built using **Microsoft Excel and XLMiner** to predict housing prices and classify affordability in the Boston housing market. The project uses a dataset of **506 housing records** and applies multiple supervised learning models to evaluate predictive performance.
